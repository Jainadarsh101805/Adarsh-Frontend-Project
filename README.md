# Visual Decision Flow Builder

Visual Decision Flow Builder is a frontend-based **workflow design application** that allows users to create and manage **decision-driven workflows** through a simple and interactive node-based interface. The application helps visualize logical flows clearly, making complex decision structures easy to design and understand.

This project focuses on frontend engineering concepts such as component-based design, state management, and intuitive UI/UX for logic-driven applications.
